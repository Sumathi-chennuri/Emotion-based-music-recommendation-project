# 🎵 Emotion-Based Music Recommendation System

## 📌 Description

This project detects user facial expressions using a webcam and recommends music based on the detected emotion.

## 😊 Emotions Detected

* Happy 😄
* Sad 😢
* Angry 😠

## ⚙️ Features

* Real-time face detection using OpenCV
* Emotion detection using smile & brightness analysis
* Automatic music recommendation 🎵
* Live camera streaming using Flask

## 🛠️ Technologies Used

* Python
* Flask
* OpenCV
* HTML, JavaScript

## ▶️ How to Run

1. Install libraries:
   pip install flask opencv-python numpy

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
